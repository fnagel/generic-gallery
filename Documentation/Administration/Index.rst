Administration
--------------


.. important::

	Please note the upgrade guide if updating from previous versions!



Installation
^^^^^^^^^^^^

* Install via EM
* Add static TS :code:`Generic Gallery: default`
* Configure TS and templates as you wish



Examples
""""""""

Add the :code:`Generic Gallery: Example - Boostrap` static TS template to your root template.
This will add a Twitter Boottsrap themed Image Slider and Teaser Slider to your predefined galleries.
